# CarRental Backend 🚗

API REST desarrollada con Laravel para la gestión de un sistema de alquiler de vehículos.

---

## ⚙️ Tecnologías

* Laravel
* PostgreSQL
* JWT Authentication

---

## 🚀 Funcionalidades

* Autenticación de usuarios (JWT)
* Gestión de vehículos
* Sistema de reservas
* Panel administrativo (endpoints)
* Estadísticas del negocio

---

## 🛠️ Instalación

git clone https://github.com/Deiver221/carrental-backend
cd carrental-backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
